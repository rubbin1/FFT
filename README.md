* （但是对font.c/h有删减，因为留存所有面积大小的字符会导致FLASH溢出，为了兼顾字符大小的可读性和没“页”屏所应该显示的内容，
* 我只保留了12x6的ASCII字符和16x16的字符串——应用了汉字取模工具，见于https://led.baud-dance.com/

* 2.对功能的说明：
* 2.1因为直接用fft计算单纯的正弦信号输入的话，有一种大炮打蚊子的感觉，所以我采用了过零检测和三点DFT进行计算频率和幅值。
* 2.1.1正弦波周期性地从负值跨越零点变为正值，每次正向过零都对应一个周期的开始，所以用第二次的时间减去第一次的时间可得周期，既而可得频率
* 2.1.2所得到的频率为粗测频率，将它换算成浮点DFT索引，真正的信号频率对应的索引是f粗 * N / f真实，所以只计算三个整数索引m−1，m，m+1处的DFT模值
* 就可以反推出真实的频率
*           ————————————该内容可见于zero_crossing_and_dft.c
* 2.2对于非正弦输入信号，采取fft进行运算，加平顶窗
* 2.2.1平顶窗：因为程序的RAM很紧张，所以如果在程序中计算平顶窗数组的话，会导致RAM溢出，所以我用python生成了一个平顶窗数组
*           ——————————该内容可见于计算平顶窗.py
* 2.2.2fft计算，就是单纯的fft计算，并把计算出的基波到五次谐波的频率和幅值存到数组里
*           ——————————该内容见于fft.c中的void fft_process_harmonics(float *freqs, float *ampls)函数
* 2.3OLED屏幕显示，有几种界面，纯正弦输入，非正弦输入的基波到五次谐波各有单独的界面，而且也包含了一个函数图像界面，可以显示一个周期的波形
*           ——————————关于汉字取模见于font.c
*           ——————————关于OLED屏的函数可见于oled.c
*           ——————————关于屏幕划线的函数，可见于void OLED_DrawLine(uint8_t x1, uint8_t y1, uint8_t x2, uint8_t y2, OLED_ColorMode color)函数
* 2.4关于函数图像绘制，在进行fft计算前，把采样到的电压值传给了Wave_Buffer[]，接着如果标志位IMAGE_MODE_ON启用，则进入void OLED_Show_Image(float f0)
* 在此函数中，我先确定了一个周期的采样点数，然后把屏幕宽度的像素点x值（一共128个）映射到一个周期的采样点上，之后再把每个采样点对应的电压值
* 也映射到像素点的y值上，这样就得到了128个坐标，把它们按顺序用OLED_DrawLine(prev_x, prev_y, x, y, OLED_COLOR_NORMAL)就能绘制图像了
* 2.5按键切换，我继承了第二轮我写的按键结构体，这样可以识别按键的短按和长按，而且我使用定时器中断溢出回调函数，
* 把TIM2定时器设为没1ms溢出，这样就会每1ms对按键进行扫描查看是否有按下的情况，在本次任务中，我用key0切换纯/非正弦输入，用key1切换非正弦输入的基波到
* 五次谐波的界面切换，用key2在非正弦输入时来切换是否到达波形图显示界面
*
* 3.整体的状态机
* ```mermaid
  stateDiagram-v2
    [*] --> 正弦模式_数据显示

    state 正弦模式_数据显示 {
        正弦_生成信号
        正弦_过零检测
        正弦_三点DFT精测
        正弦_OLED显示
    }

    state 非正弦模式_数据显示 {
        非正弦_生成信号
        非正弦_FFT加窗
        非正弦_谐波分析
        非正弦_OLED显示基波/谐波
    }

    state 非正弦模式_图像显示 {
        非正弦_生成信号
        非正弦_FFT加窗
        非正弦_谐波分析
        非正弦_OLED绘制波形
    }

    正弦模式_数据显示 --> 非正弦模式_数据显示 : KEY0 短按
    非正弦模式_数据显示 --> 正弦模式_数据显示 : KEY0 短按
    非正弦模式_图像显示 --> 正弦模式_数据显示 : KEY0 短按

    非正弦模式_数据显示 --> 非正弦模式_数据显示 : KEY1 短按 (翻页 1~5)
    非正弦模式_图像显示 --> 非正弦模式_图像显示 : KEY1 短按 (翻页 1~5)

    非正弦模式_数据显示 --> 非正弦模式_图像显示 : KEY2 短按
    非正弦模式_图像显示 --> 非正弦模式_数据显示 : KEY2 短按
  ```
* 系统启动后进入正弦模式（数据显示），执行“生成正弦波 → 过零检测 → 三点 DFT 精测 → OLED 显示频率/幅值”。
* 按下 KEY0 在两个主模式间切换（正弦 ↔ 非正弦），切换后立即执行一次对应测量流程。
* 按下 KEY1 在非正弦模式下翻页，循环显示 1～5 次谐波数据或波形图页（页码变量 pages 在 0～4 间循环）。
* 按下 KEY2 在非正弦模式下切换数据/图像显示：IMAGE_MODE_OFF 时显示数值，IMAGE_MODE_ON 时显示一个完整周期的波形图。
* 所有按键检测均在1ms定时器中断中完成消抖与标志置位，主循环轮询标志并执行相应动作，确保响应及时。
*
* 补充：
* 2026.4.26：优化了按键逻辑，纯正弦模式下，KEY1和KEY2被完全忽略，不会产生任何操作。
* 非正弦模式下，查看波形图时按KEY1不再偷偷翻页，避免回到数据界面时显示意外的谐波。
* 再次从纯正弦切回非正弦，自动重置为基波数据界面（pages = 0, 图像模式关闭），逻辑自洽且符合直觉。
